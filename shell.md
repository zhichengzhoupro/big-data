# SHELL 理解 
## shell 是什么
* shell是用C语言写的
* kenel shell user
    - kenel 用来驱动硬件干活
    - kenel 是二进制的 用户无法是直接使用的
    - shell 是命令解释器，把命令解释成二进制

# SHELL语法
## '#!' 定义脚本运行的位置
`````
#! /bin/bash
`````

## '#' 用来表示注释
``````
# description
``````

## 运行脚本
* ./  这个需要给脚本赋予权限
* bash sh 解释器直接运行 脚本不需要赋予权限

## shell 中的特殊符号
`````
~ home目录
- 上次目录
！ 执行历史命令 ！！执行上一次命令
$ 变量
& 后台执行
* 通配符 匹配所有
？通配符 匹配除回车以外的字符
；如果一行中执行多个命令
| 管道 上一个命令的输出作为下一个命令的输入
\ 转译命令 用它原本的作用而非通配符 3 \* 3
`` 命令里面执行命令  echo " Date is `date`"
'' 字符串 里面不解释变量
"" 字符串 解释变量

`````
